用iperf3测试网络吞吐量

Table of Contents

  • 1. Server side
  • 2. Client side

1 Server side

iperf3 -s 0.0.0.0

  1. In a terminal session on the first padb node (10.0.0.5) start the iperf server

2 Client side

  1. in a second terminal session on the second padb node, start the iperf client.
  1. When the test has completed, kill the iperf server using ctrl+c

./iperf3 –client node5.imlab.lan –bandwidth 0 -R –length 8900 -n 1000M –udp –port 5201 –parallel 1 –title '(1)~10Gb udp@8900: .6->.5' | tee /tmp/iperf3.$(date +%Y%m%d_%H%M).out
